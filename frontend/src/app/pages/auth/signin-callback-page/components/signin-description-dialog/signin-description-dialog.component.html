Kurze Erklärung?

Sie können jedem Teilnehmer, egal ob Produktneuheit oder Aufsteiger, bis zu 10 Punkte geben. Es existiert kein Limit, also stöbern Sie durch so viele Einträge wie Sie möchten.

Klicken Sie einfach auf einen Teilnehmer und wählen Sie in dem Drop-Down die gewünschte Punktzahl.

Vielen Dank für Ihre Stimme.
